Tecnologías utilizadas
- HTML
- CSS

- jQuery


Ejercicio 3 – Login con Ajax y PHP
Se implementó un sistema de Login utilizando una plantilla HTML descargada en formato ZIP.
La validación del usuario se realiza mediante Ajax y PHP, sin recargar la página.

Tecnologías utilizadas
- HTML
- CSS (Plantilla)
- Bootstrap
- jQuery
- Ajax
- PHP
- XAMPP
